user guide for quiz app

a guide for helping user to understand the quizapp

  1. Track the location of the user by turing on the users phone about GPS setting.
  2. Show a radius of 30m around the tracked location and load the question on to the map.
  3. When the question point is near the user winth 30m radius, users are able to answer the question by clicking the answer questionn menu and then the user can answer the relevent answer.
  4. Users can manually load them by pressing 'Get question points' in navigation menu.
  5. The user location point can be identified by clicking the central point within the blue radius circle. And When user clicked on, it will show a pop-up, displaying "You Are Here".
  6. Each question point is identified by a location name e.g. UCL library
  7. The distence between their current located position to the Warrden Street Satation can be calulated(in kilometers).
  8. The page will change to show the quiz question, once the question point is clicked -
  9. The system will then let the user know if they are right or wrong, and will show them the correct answer.
  10. When the user tries to click the question and the question location is not within the user current radius , the app will calculate the users distance.
  11. Repeat the above process for each question - until all questions are answered
